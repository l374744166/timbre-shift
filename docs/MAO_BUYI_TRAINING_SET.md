# 毛不易音色训练素材建议

目标不是让 RVC 学会完整唱法，而是让音色覆盖更稳：低声、叙述、温柔、副歌力度、明亮段落都要有一点。

## 推荐组合

1. 《像我这样的人》：低声叙述感强，适合学主体音色和口腔厚度。
2. 《消愁》：代表性强，有较完整的情绪推进，适合学副歌力度。
3. 《一荤一素》：温柔、收着唱，适合学柔和气质。
4. 《平凡的一天》：更明亮、更日常，避免模型只会沉闷低声。
5. 《借》：低声和情绪细节明显，适合补叙事型唱法质感。
6. 《给你给我》：温暖抒情，适合补中频厚度。
7. 《呓语》：更轻、更近，适合补气声和弱声细节。
8. 《盛夏》：旋律线更展开，适合补稍高一点的音区。

## 训练时注意

- 尽量用干净人声或分离质量高的人声。
- 不要只用三首很像的慢歌，容易过拟合成一种质感。
- 如果有现场版和录音室版，优先录音室版；现场混响和观众声会污染模型。
- 每首歌截取有效人声段落即可，空白、长前奏、长间奏不要太多。
- 训练后生成《恋人》这类源唱法很强的歌，仍会保留原唱节奏、气口和转音，这是 RVC 机制限制。
